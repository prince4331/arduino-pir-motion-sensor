# PIR Motion Sensor Alarm

Basic PIR motion alarm that drives an LED and buzzer while reporting activity over serial.

## Original Folder
04_Security_and_Access\PIR_Motion_Sensor

## Hardware
Arduino Uno, HC-SR501 PIR sensor, LED, buzzer.

## Notes
Threshold logic keeps the alarm latched until movement stops for stability.
